Using the Story Tale

This library automatically selects all the elements that have the "data-st-container" HTML Attribute inside the Utimate Fancy Forms.

You can easily set an text container to display in a fancy way, with all characters randomly, the "data-st-item" HTML Attribute inside the Utimate Fancy Forms.

You can add multiple effects by separating them with a comma, for example data-st-item="fadeInLeft,fadeInRight"

Each element can have it's own effect
